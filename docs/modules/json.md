# Json Tools

This page documents the `Json Tools` available in CVTools.

## clean_labels_json

**Class:** `CleanLabelsJsonTask`

Task for cleaning specific fields in JSON label files.

    This class extends the base Task class to modify JSON files by setting
    specified fields to given values, typically used for cleaning up
    unnecessary data like imageData fields.

    Attributes
    ----------
    params : object
        Configuration parameters including folder_path, field, and value.

Example YAML:
    ```yaml
- name: clean_labels_json
  params:
    folder_path: <value>
    field: <value>
    value: <value>
    ```

### YAML example

    ```yaml
- name: clean_labels_json
  params:
    folder_path: <value>
    field: <value>
    value: <value>
    ```

## fix_labelme_json

**Class:** `FixLabelmeJsonTask`

Task for fixing LabelMe JSON files to be fully valid and openable in LabelMe.

    This class extends the base Task class to correct JSON files generated by
    auto_label_labelme.py, ensuring they are compatible with LabelMe software.

    Attributes
    ----------
    SHAPE_DEFAULTS : dict
        Default values for shape fields in LabelMe JSON.
    IMG_EXTENSIONS : tuple
        Supported image file extensions.

Example YAML:
    ```yaml
- name: fix_labelme_json
  params:
    labels_dir: <value>
    images_dir: <value>
    version: <value>
    dry_run: <value>
    ```

### YAML example

    ```yaml
- name: fix_labelme_json
  params:
    labels_dir: <value>
    images_dir: <value>
    version: <value>
    dry_run: <value>
    ```
